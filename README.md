# Spotify Data Analysis

This project analyzes Spotify music data to discover insights about artists, popularity, and track features using Python.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Insights
• Most popular artists
• Audio feature correlations
• Track popularity distribution

## Dataset
Spotify Tracks Dataset
